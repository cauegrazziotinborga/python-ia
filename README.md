# ChatBot com Django + LangChain + Groq

---

Aluno: Cauê Grazziotin Borga

Trabalho desenvolvido utilizando:

- Python
- Django
- LangChain
- Groq API
- Bootstrap
- BeautifulSoup

O sistema permite:

- Informar uma URL como fonte de informação
- Realizar perguntas com base no conteúdo do site
- Visualizar respostas em interface estilo chat
- Exibir histórico da conversa
- Limpar histórico da conversa
